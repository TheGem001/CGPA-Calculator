# 🎓 MyIUB CGPA Calculator

**Official GPA Logic for The Islamia University of Bahawalpur**

![Version](https://img.shields.io/badge/version-1.0-blue)
![Status](https://img.shields.io/badge/status-inactive-red)

A specialized tool for IUB students to calculate GPA based on the official **IUB Ready Recknor** table. Specifically pre-configured for the Software Engineering department.

🔗 **Launch App:** [https://thegem001.github.io/CGPA-Calculator/](https://thegem001.github.io/CGPA-Calculator/)

---

## 🚀 Key Features

* **Official Grading:** Uses the exact IUB Marks-to-GP conversion (e.g., 85+ = 4.0).
* **Dynamic Courses:** Pre-loaded with Semester 1 subjects; easily add or remove custom courses.
* **Instant Results:** Real-time GPA calculation with performance color-coding.
* **Reference Table:** Built-in grading guide for quick verification.

---

## 💻 Tech Stack

* **Core:** React 18 & Tailwind CSS
* **Deployment:** GitHub Pages
